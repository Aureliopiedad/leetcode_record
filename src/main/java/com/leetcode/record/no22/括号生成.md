# No 22 括号生成

## 题目

数字 n 代表生成括号的对数，请你设计一个函数，用于能够生成所有可能的并且 有效的 括号组合。

### 示例 1

输入：n = 3
输出：["((()))","(()())","(())()","()(())","()()()"]

### 示例 2

输入：n = 1
输出：["()"]

## 思路

一眼回溯，因为左括号一定先产生，所以先考虑全部左括号+全部右括号，再依次将左括号替换成右括号：

```shell
((())) l+=3 r+=3
(())() l+=2 r+=2 l+=1 r+=1
(()()) l+=2 r+=1 l+=1 r+=2
()(()) l+=1 r+=1 l+=2 r+=2
()()() l+=1 r+=1 l+=1 r+=1 l+=1 r+=1
```
